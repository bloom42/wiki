# Development

Bloom use the following platforms to develop it's products:

* [Go](#go)
* [NodeJS](#nodejs)
* [Flutter](#flutter)

## Go

1. Install [Go](https://golang.org/)

2. Install [gowatch](https://gitlab.com/bloom42/tools/gowatch)
```sh
$ go install gitlab.com/bloom42/tools/gowatch
```
